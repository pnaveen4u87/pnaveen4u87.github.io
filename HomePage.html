About the Visualization: For this Gun Violence analysis project, I have used 2014 to 2017 calendar year U.S Gun Violence data set from Kaggle(Gun Violence data) and 2014 to 2017 calendar year U.S population data set from Census(Population data). I have chosen the factors like Number of incidents, year and used US population data to show the rate of Gun Violence per 1000 population for the years 2014 - 2017.

This project is based on hybrid interactive slide show. Guidance is provided on each slide, but the users are free to drill down and list some details that they might be interested on or they can choose just to continue.

Scenes
Each scene is a single chart with a specific goal related to the same subject and in a specific order to best convey the message. Each slide has similar template, fonts, hue and tool tip to provide visual consistency on scene.

Annotations
Annotations are provided in appropriate places in each chart, which is highlighted in different color and a tooltip, when mouse is hovered over the data points of each chart. The legend and labels along the X and Y axis provides additional annotations for the chart.

Triggers
The menu items along with the prev and next button in the navigation bar of each page enable the User Interface events. The mouse click acts as a trigger event for navigation. User interacts by clicking on the button or menu items to navigate to various part of this story. The mouse hover is another user interface event. The hover event triggers the display of tooltip in all the line and bar charts.

Parameters
The name of the target slide is passed as an input paramter to the button and navigation bar items. Based on the user selection, appropriate data is filtered and respective slide with appropriate chart for the scene is displayed. The active slide is highlighted in green will let user know which page they are in.

Using the links under Discussed Visualization user can traverse to respective graphs to find overall incidents reported across USA between 2014 and 2017, then, which state has highest or lowest Gun Violence Incidents and which state has highest rate of Gun Violence per 1000 population which gives you an overview of which states are safe and which states are dangerous.
